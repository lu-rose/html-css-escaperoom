Wrong Color in Component

Open Reported by: Designer · Color inconsistency

The designer flagged a color inconsistency in the component library. One swatch in the grid below is using a slightly different green than all the others — it slipped through because it looks almost the same at a glance. Find the odd one out and enter its exact hex code (without #).

Component library - squares

Open the Inspector cursor (the arrow icon in DevTools, or press Ctrl+Shift+C / Cmd+Shift+C). Hover over each swatch and watch the Styles pane for its background-color. Most will show the same value. The one that is different is the answer — enter its hex code without the # symbol.
