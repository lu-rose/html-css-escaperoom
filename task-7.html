Bug Report 07 / 14

The button text is invisible

Open Reported by: Designer · CSS · Color contrast · WCAG · Blocker

The "Send message" button is technically present, but the text is invisible. Users see a box but have no idea what it does. The text and background are nearly the same shade of grey — contrast ratio is only ~1.05:1. WCAG AA requires at least 4.5:1 for body text. Find the broken color in DevTools, then agree as a team on a fix. Enter a hex color for the text that passes WCAG AA.

Contact form — inspect this

Step 1: In DevTools → Elements, select the button. In Styles, find color: #f2f2f2 — almost identical to the background #f0f0f0.

Step 2: Chrome DevTools has a built-in contrast checker. Click the colour swatch next to the color value in Styles and scroll down — it shows the current contrast ratio.

Step 3: Pick any color with ratio ≥ 4.5:1. Dark grey or black always works — try #1c1f3d or #333333.
