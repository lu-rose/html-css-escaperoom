Bug Report 14 / 20

Distracting Animation

Open Reported by: Users · Animation interfering with content

A spinning animation was added to a status indicator — but it is running so fast that the text inside it is completely unreadable. Users cannot get the information they need. Stop the animation and read what it says.

REFACTOR

Pause the animation first — then the answer field will unlock.

Level 1: Start by inspecting the key element in this task and look for the one property or attribute that is causing the bug.

Still stuck? Show Level 2
In the Elements panel, select #spinning. In the Styles pane, click the area where you can add a new declaration and type:

animation-play-state: paused

This freezes the animation so you can read the text. What it says is your answer.
