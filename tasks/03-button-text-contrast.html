Bug Report 03 / 14

The button text is invisible

Open Reported by: Designer · CSS · Color contrast · WCAG · Blocker

The "Send message" button is technically present, but the text is invisible. Users see a box but have no idea what it does. The text and background are nearly the same shade of grey. In DevTools, edit either the button text color or the button background color until the contrast passes WCAG AA. When the ratio is good enough, a continue button will appear automatically.

Official reference: WCAG 2.2 — Contrast (Minimum).

Contact form — inspect the button

Still stuck? Show Level 2
Step 1: In DevTools → Elements, select the button. In Styles, find color: #f2f2f2 — almost identical to the background #f0f0f0.

Step 2: Chrome DevTools has a built-in contrast checker. Click the colour swatch next to the color value in Styles and scroll down — it shows the current contrast ratio.

Step 3: Pick any color with ratio ≥ 4.5:1. Dark grey text or a much darker/lighter background always works. When the ratio passes, the next button appears automatically.
