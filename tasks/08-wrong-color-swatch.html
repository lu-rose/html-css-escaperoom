Wrong Color in Component

Open Reported by: Designer · Color inconsistency

The designer flagged a color inconsistency in the component library. One swatch in the grid below is using a slightly different green than all the others. It slipped through because it looks almost the same at a glance. Find the odd one out and enter its exact hex code.

Component library - squares

Level 1: Start by inspecting the key element in this task and look for the one property or attribute that is causing the bug.

Still stuck? Show Level 2
Open the Inspector cursor (the arrow icon in DevTools, or press Ctrl+Shift+C / Cmd+Shift+C). Hover over each swatch and watch the Styles pane for its background-color. Most will show the same value. The one that is different is the answer — enter its hex code, e.g. #FFFFFF.
