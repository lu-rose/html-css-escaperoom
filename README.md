# Fagdag 08.05.2025
A browser-based escape room built for Fagdag 08.05.2026, where participants step into the shoes of a developer. Players work through 10 bug reports using only their browser's DevTools — inspecting elements, fixing CSS, and uncovering hidden content. No installation needed; open `index.html` in a browser to play.

This is vibe-coded 👨‍💻
The site is live at [https://lu-rose.github.io/html-css-escaperoom/](https://lu-rose.github.io/html-css-escaperoom/)
